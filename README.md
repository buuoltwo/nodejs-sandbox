# express
## 构建：
1. nvm安装node.js 
2. npm i -g express-generator
3. 使用express-generator生成一个express项目
 ```
    express --view=ejs what_i_love #项目名
    cd what_i_love
    npm i
 ```
4. npm install -g eslint
